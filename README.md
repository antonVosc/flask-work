# flask-work
This is a repository for learning Flask.

To run code, enter:

1. 
```
git clone https://github.com/antonVosc/flask-work.git
```

2. 
```
cd flask-work/
```

3. 
```
python3 -m venv env
```

4. 
```
. env/bin/activate
```

5. 
```
pip install -r requirements.txt
```

6. 
```
export FLASK_APP=market.py
```

7. 
```
flask run
```
